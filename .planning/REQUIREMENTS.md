# Requirements: Regionalprojekt (Ennstal Aktuell)

**Defined:** 2026-03-25
**Core Value:** Steiermark residents get relevant, hyperlocal news for their Bezirk — automatically, without an editorial team needed to run it.

## v1.1 Requirements

Requirements for the Design Overhaul milestone. Each maps to roadmap phases.

### Design System

- [x] **DS-01**: Site uses Newsreader for headlines, Inter for body text, Work Sans for labels/uppercase text
- [x] **DS-02**: Color palette uses Styrian green (#2D5A27), warm cream (#fbfaee), alpine red (#8b0000), muted sage (#4a5d4e)
- [x] **DS-03**: Icons use Material Symbols Outlined throughout the reader frontend
- [x] **DS-04**: Border radius defaults to 2px (sharp editorial corners)

### Header & Identity

- [ ] **HDR-01**: Fixed Styrian identity bar (white/green horizontal stripe) at top of viewport
- [ ] **HDR-02**: Dark green header with italic serif "RegionalNews" branding
- [ ] **HDR-03**: Steiermark location badge in header
- [ ] **HDR-04**: Search icon in header linking to search page

### Homepage

- [ ] **HOME-01**: Full-bleed hero featured article with image, gradient overlay, category badge, and large serif headline
- [ ] **HOME-02**: Horizontally scrollable "Top-Meldungen" cards with bottom border accent
- [ ] **HOME-03**: Topic sections with editorial grid layout, wood dividers, and Styrian flag accents
- [ ] **HOME-04**: Breaking news "Eilmeldung" red banner when an article is flagged

### Article Detail

- [ ] **ART-01**: Article page uses editorial typography (Newsreader headlines, Inter body)
- [ ] **ART-02**: Article page uses warm cream background with consistent design system styling

### Search & Categories

- [ ] **SRCH-01**: Search page with large serif search input for filtering articles
- [ ] **SRCH-02**: Trending topics displayed as pill-shaped tags
- [ ] **SRCH-03**: Category grid with hover states for browsing by topic
- [ ] **SRCH-04**: Recommended articles section below categories

### Bottom Navigation

- [ ] **NAV-01**: Bottom nav restyled with warm cream background, updated Material Symbols icons
- [ ] **NAV-02**: Active nav item displays as filled pill with primary green background

## Future Requirements

### Reader Features (deferred)

- **SAVE-01**: User can bookmark/save articles ("Gemerkt" functionality)
- **PROF-01**: User profile/account page ("Profil/Konto" functionality)
- **TREND-01**: Trending topics based on real analytics/engagement data

## Out of Scope

| Feature | Reason |
|---------|--------|
| Dark mode | Not in mockups, defer to future milestone |
| CMS/admin redesign | v1.1 focuses on reader frontend only |
| New content types | Design only, no new data models beyond Eilmeldung flag |
| Animations/transitions | Beyond hover states shown in mockups |
