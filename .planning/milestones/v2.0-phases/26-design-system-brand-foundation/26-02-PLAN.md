For EVERY reader component file, apply these systematic replacements:

**1. Token migration (search-and-replace across all files):**
- `styrian-green` → `primary` (in all Tailwind class contexts: bg-, text-, border-, ring-, etc.)
- `cream-dark` → `surface`

**2. Border removal (the core visual change):**
For each border occurrence, replace with the appropriate tonal alternative:

- **Card borders** (`border border-cream-dark`, `border border-sage/30`): Remove entirely. Cards get `bg-surface-elevated shadow-sm` instead. The shadow provides subtle depth without a visible line.
- **Divider lines** (`border-t`, `border-b`, `border-0 h-px`): Replace with spacing only (`my-6` or `py-4`) or tonal background shift. For list separators, use `bg-surface` on alternate items or add padding with no visual line.
- **Input/form borders** (`border-2 border-styrian-green`, `border border-zinc-200`): Replace with `bg-surface ring-1 ring-secondary/20 focus:ring-primary` — tonal fill with subtle ring.
- **Navigation borders** (`border-t border-styrian-green/10`): Replace with `shadow-[0_-1px_3px_rgba(0,0,0,0.08)]` (top shadow) or just tonal background difference.
- **Active state borders** (`border-styrian-green`): Replace with `bg-primary/10 text-primary` (tonal highlight, no border).
- **Button borders** (`border border-sage/30`, `border border-zinc-300`): Replace with `bg-surface-elevated shadow-sm` for secondary buttons. For primary CTAs: see CTA gradient below.
- **Spinner border** in ArticleFeed.tsx: Keep as-is (functional animation border, not decorative).

**3. Corner radius:**
- All `rounded-sm` already maps to 0.75rem (from @theme override in Plan 01).
- Any `rounded-none` or missing rounded on interactive elements: add `rounded-sm`.
- CTA buttons: use `rounded-full` (pill shape).
- Cards: use `rounded-sm` (0.75rem).

**4. CTA gradient buttons:**
Any primary action button (e.g., "Mehr laden" in SearchPageLayout, submit buttons) gets:
`bg-gradient-to-br from-primary to-primary-container text-white rounded-full px-6 py-2.5 font-semibold shadow-sm`
(135deg = to-br in Tailwind)

Secondary/ghost buttons get: `bg-surface-elevated text-primary shadow-sm rounded-sm`

**5. Article detail page** (`artikel/[publicId]/[slug]/page.tsx`):
- Replace any legacy color tokens with semantic equivalents.
- Ensure article body container uses `bg-background` canvas.
