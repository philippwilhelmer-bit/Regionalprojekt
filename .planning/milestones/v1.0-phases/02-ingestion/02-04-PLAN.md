<feature>
  <name>Generic RSS/Atom feed adapter</name>
  <files>src/lib/ingestion/adapters/rss.ts, src/lib/ingestion/adapters/rss.test.ts</files>
  <behavior>
    rssAdapter(source: Source) → Promise<RawItem[]>:
    - Fetches source.url with native fetch()
    - Throws Error on non-200 response
    - Passes response text to feedsmith parseFeed()
    - Maps each item/entry to RawItem:
      - externalId: item.guid ?? item.link ?? computeContentHash(item.title ?? '', item.description ?? '')
      - title: item.title ?? ''
      - body: item.description ?? item.content ?? ''
      - publishedAt: item.pubDate ? new Date(item.pubDate) : null
      - sourceUrl: item.link ?? source.url
      - rawPayload: original item object from feedsmith
    - For Atom feeds: entry.id → externalId, entry.title → title, entry.summary → body, entry.published → publishedAt
    - Returns empty array (not throw) if feed parses but has 0 items
    - Tests use fixture XML files — no real HTTP calls
  </behavior>
  <implementation>
    Install feedsmith first: `npm install feedsmith`
    Use parseFeed universal parser — it auto-detects format. The returned object is a discriminated union by format; handle both RSS items and Atom entries by checking feed format or accessing common normalized fields.
    The externalId fallback to computeContentHash ensures feeds without guids still deduplicate correctly.
  </implementation>
</feature>
