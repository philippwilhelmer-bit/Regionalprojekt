  <behavior>
    Types (types.ts — no tests needed, pure interface):
    - RawItem: { externalId: string; sourceUrl: string; title: string; body: string; publishedAt: Date | null; rawPayload: unknown }
    - AdapterFn: (source: Source) => Promise<RawItem[]>

    computeContentHash(title, body):
    - Returns SHA-256 hex string
    - Normalizes: lowercase, collapse whitespace to single space, trim, join with "||"
    - Same result for "  Hello  World  " and "hello world" in same position
    - Different result for different title or body

    isDuplicate(db, source, externalId, contentHash):
    - Fast path: findFirst where { source, externalId } — returns true if found
    - Slow path: findFirst where { contentHash } — returns true if found
    - Returns false if neither match exists
    - Uses TypeScript overload pattern for DI (duck-type $connect check)
  </behavior>

       - isDuplicate uses TypeScript overloads: (source, externalId, contentHash) for prod, (client, source, externalId, contentHash) for tests
       - Duck-type check: '$connect' in firstArg
