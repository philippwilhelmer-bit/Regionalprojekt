  <behavior>
    otsAtAdapter(source: Source) → Promise<RawItem[]>:
    - Fetches /api/liste with app key, anz=50, von=(24 hours ago unix timestamp)
    - For each list item: checks if externalId (OTSKEY) already exists in Article table by source+externalId
    - Fetches /api/detail ONLY for items not already in DB
    - Maps each detail response to RawItem: externalId=OTSKEY, title=TITEL, publishedAt=new Date(ZEITSTEMPEL*1000), sourceUrl=WEBLINK, rawPayload=raw detail response object
    - Body field: try common candidates (TEXT, BODY, INHALT, text, body) with null-safe access; fall back to empty string; log warning if none found
