<!-- Source health dashboard data shape from sources-actions.ts (Plan 04) -->

listSourcesAdmin() returns sources with:
```typescript
{
  id: number
  url: string
  type: ArticleSource  // OTS_AT | RSS | MANUAL
  enabled: boolean
  pollIntervalMinutes: number
  healthFailureThreshold: number
  healthStatus: SourceHealth  // OK | DEGRADED | DOWN
  lastSuccessAt: Date | null
  latestRun: { itemsFound: number, itemsNew: number, createdAt: Date } | null
  failedErrorCount: number  // FAILED + ERROR article count for this source type
}
```

Health badge colors (Tailwind):
- OK → green badge (bg-green-100 text-green-700)
- DEGRADED → yellow badge (bg-yellow-100 text-yellow-700)
- DOWN → red badge (bg-red-100 text-red-700)

<!-- AiConfig form values -->
AiConfig fields: tone (NEUTRAL/FORMAL/CONVERSATIONAL), articleLength (SHORT/MEDIUM/LONG),
styleNotes (textarea, optional), modelOverride (text, optional)

PipelineConfig fields: maxRetryCount (number), deadManThresholdHours (number)

<!-- Server Action wrappers -->
ai-config-actions.ts needs 'use server' + requireAuth() + call to DAL:
```typescript
'use server'
import { getAiConfig as getAiConfigDal, upsertAiConfig } from './ai-config-dal'
export async function upsertAiConfigAction(formData: FormData) {
  await requireAuth()
  return upsertAiConfig(/* parse from formData */)
}
```

Same pattern for upsertAiSourceConfigAction, deleteAiSourceConfigAction, upsertPipelineConfigAction.
