**Create `src/components/reader/BottomNavClient.tsx`** — new file, `"use client"` directive:

```typescript
"use client";

import Link from "next/link";
import { usePathname } from "next/navigation";
```

Define `NAV_ITEMS` as a const array with four items:
1. `{ href: "/", icon: "newspaper", label: "Nachrichten", enabled: true }`
2. `{ href: "/suche", icon: "search", label: "Suche", enabled: true }`
3. `{ href: "#", icon: "bookmark", label: "Gemerkt", enabled: false }`
4. `{ href: "#", icon: "person", label: "Profil", enabled: false }`

Export `function BottomNavClient()`:
- Call `const pathname = usePathname();`
- Render `<nav>` with classes: `fixed bottom-0 inset-x-0 z-40 bg-cream border-t border-sage/10 h-16 flex items-center justify-around px-2`
  - Note: subtle `border-sage/10` replaces the old zinc border-top (per CONTEXT.md: "subtle sage or no border")
- Map over NAV_ITEMS. For each item:
  - Determine `isActive`: For enabled items only. If `href === "/"`, use exact match `pathname === "/"`. Otherwise use `pathname.startsWith(item.href)`.
  - **Disabled items** (`enabled: false`): Render as `<div>` (not Link, not button) with `className="flex flex-col items-center gap-0.5 opacity-40 cursor-default"`. Inside: Material Symbols icon span (`text-xl text-sage`) + label span (`font-label text-xs text-sage`).
  - **Enabled items**: Render as `<Link href={item.href}>` with `className="flex flex-col items-center gap-0.5 min-w-[56px]"`. Inside:
    - Pill wrapper: `<span className="flex items-center justify-center w-16 h-8 rounded-full ${isActive ? 'bg-styrian-green' : ''}">`
      - Icon: `<span className="material-symbols-outlined text-xl ${isActive ? 'text-white' : 'text-sage'}" aria-hidden="true">{item.icon}</span>`
    - Label: `<span className="font-label text-xs ${isActive ? 'text-styrian-green font-medium' : 'text-sage'}">{item.label}</span>`
