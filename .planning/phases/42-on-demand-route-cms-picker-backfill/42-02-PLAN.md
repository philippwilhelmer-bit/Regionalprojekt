    Render:
    - If savedUrl: show img preview (same styling as UnsplashPicker — `w-full h-40 object-cover rounded border border-surface`), credit text below, "Entfernen" button (absolute top-right, red).

    - Status message if message is non-empty: `text-sm text-ink-dim`.

    Props: `{ articleId: number; headline: string; currentImageUrl: string | null; currentImageCredit: string | null }`

To avoid modifying UnsplashPicker, wrap it in a div and accept the double label OR simply accept it as is — the tab makes the context clear enough.
