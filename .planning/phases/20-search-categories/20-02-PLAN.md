    **1. Create `src/components/reader/SearchPageLayout.tsx`** — a `"use client"` component.

    Props interface:
    ```typescript
    interface SearchPageLayoutProps {
      articles: ArticleWithBezirke[]
      bezirke: Bezirk[]
      recommended: ArticleWithBezirke[]
    }
    ```

    State:
    - `query: string` (useState, default "")
    - `activeBezirkId: number | null` (useState, default null)

    Derived (useMemo):
    - `isFiltered = query.trim() !== "" || activeBezirkId !== null`
    - `filtered` — filter `articles` array:
      - If `activeBezirkId` set: keep articles where `a.bezirke.some(entry => entry.bezirkId === activeBezirkId)`
      - If `query` not empty: keep articles where `a.title?.toLowerCase().includes(query.toLowerCase())`
      - Both combine as AND filter
    - Dependencies: `[articles, query, activeBezirkId]`

    Layout (4 vertically stacked zones on cream background `bg-cream min-h-screen`):

    **Zone 1 — Search input** (SRCH-01):
    - Full-width container with `px-4 pt-6 pb-4`
    - Heading: "Suche" in `font-headline text-3xl text-zinc-900 mb-4`
    - `<div className="relative">` wrapping the input
    - `<span className="material-symbols-outlined absolute left-0 top-1/2 -translate-y-1/2 text-sage">search</span>` as prefix icon
    - `<input type="text">` (NOT type="search" — avoids browser-native clear button inconsistency) with:
      - `value={query}` + `onChange={e => setQuery(e.target.value)}`
      - `placeholder="Artikel suchen…"`
      - Classes: `w-full font-headline text-xl bg-transparent border-b-2 border-sage/40 focus:border-styrian-green outline-none pb-2 pl-8 pr-8 text-zinc-900 placeholder:text-zinc-400`
    - When `query !== ""`: render a clear X button on the right side — `<button onClick={() => setQuery("")}>` with Material Symbols `close` icon, absolute positioned right-0

    **Zone 2 — Trending pills** (SRCH-02):
    - `<section className="px-4 pb-4">`
    - Heading: `<h2 className="font-label text-xs uppercase tracking-wider text-sage mb-2">Beliebte Themen</h2>`
    - Horizontal scroll row: `<div className="flex gap-2 overflow-x-auto scrollbar-none pb-2">`
    - Each Bezirk as a `<button>` with `shrink-0`:
      - `onClick={() => setActiveBezirkId(b.id === activeBezirkId ? null : b.id)` (toggle)
      - Active: `bg-styrian-green text-white`
      - Inactive: `bg-white border border-sage/30 text-sage hover:border-styrian-green hover:text-styrian-green`
      - Common: `px-3 py-1 rounded-full text-sm font-label transition-colors`

    **Conditional rendering based on `isFiltered`:**

    If `isFiltered` is TRUE — show search results:
    - Result count: `<p className="px-4 pb-2 text-sm text-sage">{filtered.length} Artikel gefunden</p>`
    - If `filtered.length === 0`: show no-results state:
      - `<div className="px-4 py-12 text-center">`
      - `<p className="font-headline text-xl text-zinc-600 mb-2">Keine Artikel gefunden</p>`
      - `<p className="text-sm text-zinc-400">Versuchen Sie einen anderen Suchbegriff oder wählen Sie einen Bezirk.</p>`
    - If results exist: 2-column card grid:
      - `<div className="grid grid-cols-2 gap-3 px-4 pb-6">`
      - Map `filtered` to `<ArticleCard article={a} key={a.id} />`

    If `isFiltered` is FALSE — show discovery zones:

    **Zone 3 — Category grid** (SRCH-03):

    - Heading: `<h2 className="font-label text-xs uppercase tracking-wider text-sage mb-3">Alle Bezirke</h2>`
