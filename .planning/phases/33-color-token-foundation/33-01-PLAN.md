- REMOVE the existing --radius-sm: 0.75rem override entirely. Tailwind v4 defaults (--radius-xs: 0.125rem, --radius-sm: 0.25rem) already match our target values.

CRITICAL pitfall: Do NOT use var() inside color-mix() expressions within @theme. Tailwind v4 processes @theme statically. Use literal hex values (#071806, #FCF9EF) inside all color-mix() calls.
