## Decisions Made

- `buildArticleMetadata()` returns `{}` on null article — allows `generateMetadata()` to call it without an intermediate null check, relying on Next.js's notFound() in the page component for proper 404 handling.
- `ShareButton` extracted as a separate `'use client'` component — keeps the main article detail page as a Server Component, maximizing RSC benefits.
- JSON-LD inlined via `dangerouslySetInnerHTML` as specified in RESEARCH.md Pattern 5 — no external schema library needed for a single structured data type.
- Related articles filtered after fetching (Array.filter) rather than using a Prisma `NOT` clause — simple and correct for a max-5 result set.
