<objective>
Wire the (public) layout to fetch Bezirke from the DB once and pass them as props to Header and BezirkModal, removing the hardcoded arrays from both Client Components.

Purpose: The layout is a React Server Component — it can call `listBezirke()` directly. Both Header and BezirkModal are `"use client"` components; they cannot fetch data themselves. Prop-passing from Server to Client Component is the established pattern in this codebase (used on article pages). This removes the last hardcoded UI region lists, making the UI automatically reflect any future `config.regions` + seed change.

Output: Async `(public)/layout.tsx` that calls `listBezirke()` once; `Header.tsx` with `bezirke` prop replacing `BEZIRK_NAMES`; `BezirkModal.tsx` with `bezirke` prop replacing `BEZIRKE`.
</objective>
