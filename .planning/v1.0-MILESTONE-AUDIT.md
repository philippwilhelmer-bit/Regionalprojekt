## Defect Detail (Closed)

The previous audit (before phases 11 and 12) found 3 defects. All have been closed:

### Defect 1 (Medium) — CLOSED by Phase 11
**`isStateWide` never set by AI pipeline**
`pipeline.ts` now detects `'steiermark-weit'` in `step1.bezirkSlugs` (line 125) and writes `Article.isStateWide=true` (line 138). Three new integration tests confirm the fix.

### Defect 2 (Low) — CLOSED by Phase 12
**Region list hardcoded in UI**
`BezirkModal.tsx` and `Header.tsx` now receive `bezirke: BezirkItem[]` as a prop from `(public)/layout.tsx`, which calls `listBezirke()` from the database. No hardcoded arrays remain.

### Defect 3 (Very low) — CLOSED by Phase 12
**`features.rss` flag not enforced**
`/rss/[slug]/route.ts` now checks `config.features.rss` as the first statement before any DB access.
